My Mini-project

Summery

For my mini-project, I am going to write a script that finds set phrases or N-Grams from a given dataset and displays their frequency. The dataset that I have chosen for my project is the BAWE Corpus which contains various written texts in academic English. The set phrases I’ll be using will be taken from the Academic Formulas List (AFL) which is a list of the most common formulaic sequences in academic English. The research question for this project is: “Do the topmost common formulaic sequences in the AFL also appear in BAWE, and if so, what are their frequencies?”
